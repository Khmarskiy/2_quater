CSS свойство объекта:
display: отвечает за отображение элемента. 
    none - не показывает элемент
    block - делает элемент блочным (есть ширина высота, но размещаются друг под другом)
    inline- 
    inline-block - 
    table-cell - элемент таблицы
    flex - превращает объект во flex(гибкий) элемент

flex-wrap: wrap - делает flex объект перемещаемым на новую строку, если он не помещается в текущей
justify-content: 
    flex-start (по умолчанию) - выравнивает блоки по левому краю (если не включен поворот оси)
    flex-end - выравнивает блоки по правому краю.(если не включен поворот оси)
    center - выравнивает блоки по центру
    space-around - делает между блоками в одной строке одинаковые расстояния. не прижимает блоки к краю контейнера.
    space-between - равномерно растягивает пространство между блоками.(крайние блоки прижаты к границе контейнара)\
aligne-items: 


aligne-self: - выравнивание блока по вертикали. 
    center;
flex-direction: - ориентация основной оси блоков контейнера.
    column - сверху вниз 

flex-grow: x - соотношение элемента в контейнере к другим элементам. 
flex-basis - минимальный размер блока. 
max-width - максимальный размер контейнера.
order - указывает порядок блока.
